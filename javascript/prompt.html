例

上の例は次のように書きました。

<p id="greet"></p>

<script>
let s = prompt('お名前は？');
if (s != null && s != '') {
  document.getElementById('greet').textContent = 'こんにちは，' + s + 'さん！';
}
</script>

ここで textContent を innerHTML にすればタグも解釈するようになります（危険です）。
