日本の超過死亡？

[2021-10-15] 2021年5月までの概数，7月までの速報を追加した。

厚労省人口動態調査から，人口動態統計（確定数），人口動態統計月報（概数），人口動態統計速報がリンクされている。確定数は翌年9月ごろにならないと出ない。概数はほぼ5ヶ月後，速報はほぼ2ヶ月後に出る。

データはe-Statの人口動態調査で公開されている。ここから集めた概数・速報データを japandeaths.csv というファイルに収めた。

import matplotlib.pyplot as plt
import pandas as pd
import numpy as np
import datetime

def days(year, month):
    year = int(year)
    month = int(month)
    m = [31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31]
    return m[month - 1] + ((month == 2) and
                           ((year % 4 == 0) and (year % 100 != 0) or (year % 400 == 0)))

df = pd.read_csv('https://oku.edu.mie-u.ac.jp/~okumura/python/data/japandeaths.csv')
t = [datetime.datetime(int(x['年']), int(x['月']), 1) for i, x in df.iterrows()]

perday1 = np.array([r[2] / days(r[0], r[1]) for i, r in df.iterrows()])
perday2 = np.array([r[3] / days(r[0], r[1]) for i, r in df.iterrows()])

plt.plot(t, perday1, 'o-')
plt.plot(t, perday2, 's-')
plt.ylabel('1日あたり死亡数')
plt.legend(['概数', '速報値'])
plt.savefig('japandeaths1.svg', bbox_inches="tight")

for y in range(2014, 2022):
    plt.plot(df[df['年'] == y]['月'], perday1[df['年'] == y],
             alpha=0.5, marker=f'${y % 10}$', label=y)
plt.plot(df[df['年'] == 2021]['月'], perday2[df['年'] == 2021],
         'o-k', label='2021速報値')
plt.xlabel('月')
plt.ylabel('1日あたり死亡数')
plt.legend(loc=(0.45,0.48))
plt.savefig('japandeaths2.svg', bbox_inches="tight")

perday = np.array([p2 if np.isnan(p1) else p1 for p1, p2 in zip(perday1, perday2)])
for m in range(1, 13):
    plt.plot(df[df['月'] == m]['年'], perday[df['月'] == m], marker=f'${m}$')
plt.ylabel('1日あたり死亡数')
plt.savefig('japandeaths3.svg', bbox_inches="tight")
