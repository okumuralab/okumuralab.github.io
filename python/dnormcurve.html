正規分布の密度関数を描く

正規分布の密度関数を描いてみましょう。 

import matplotlib.pyplot as plt
import numpy as np

x = np.linspace(-3, 3, 61)  # 区間[-3,3]を60等分する61点
y = np.exp(-x**2 / 2) / np.sqrt(2 * np.pi)
plt.plot(x, y)

x ≧ 1 の部分だけ塗りつぶしてみましょう。

plt.fill_between(x[x >= 1], y[x >= 1])
