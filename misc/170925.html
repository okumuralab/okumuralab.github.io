CentOS 7でRuby 2.4を最速インストール

最新のgemがCentOS 7標準のRubyでインストールできない！ そこで，Ruby 2.4を，従来の環境を壊さず最速でインストールする：

yum -y install centos-release-scl-rh centos-release-scl
yum --enablerepo=centos-sclo-rh -y install rh-ruby24 rh-ruby24-ruby-devel
scl enable rh-ruby24 bash

新しいRubyを使うための環境を整えた新しい bash が起動する。ここで gem install ... などをし，最後に exit して元の環境に戻る。

cronで動くスクリプトなどについては，

/usr/bin/scl enable rh-ruby24 -- bash -c 'コマンド'

のようにして実行すればよいであろう。
