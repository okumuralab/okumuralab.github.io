モンテカルロ法でeを求める

モンテカルロ法でπを求めるのは，よく見かける（例）。eは [0,1) 一様分布の乱数の和が1を超える個数の期待値として求められる（Twitterで教えていただいた証明例：一様分布の和の平均到達時間 ―eを近似する確率シミュレーション―，18th Putnam 1958 Problem A3）。

Pythonで書いてみよう：

import random

def ne():
    s = 0
    i = 0
    while s < 1:
        s += random.random()
        i += 1
    return i


n = 1000000
t = 0
for _ in range(n):
    t += ne()
print(t / n)

ほぼ 2.718 となる。
