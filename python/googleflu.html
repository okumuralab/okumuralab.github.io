Google Flu Trends のプロット

[2023-03-06] Google Flu Trends は現在は Google から消えています。以下では Internet Archive のURLも挙げておきます。

Google Flu Trends（Wikipedia 記事）は Google が検索入力からインフルエンザの流行を調べたプロジェクトである。予測はあまり正確でなく，2015年に打ち切られた。当時の週ごとの予測データは https://www.google.org/flutrends/about/ で公開されていた（今は消えているが こちら にアーカイブがある）。

日本についてのデータを読んでプロットしてみよう（データもGoogleから消えているがアーカイブが こちら にある）：

import pandas as pd
import matplotlib.pyplot as plt

df = pd.read_csv("https://www.google.org/flutrends/about/data/flu/jp/data.txt",
                 index_col='Date', parse_dates=['Date'],
                 skiprows=11)
df.plot()
plt.savefig('googleflu-jp.png', bbox_inches="tight")
