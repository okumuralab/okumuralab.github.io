自殺統計

自殺統計はe-Statには新しいものが見つからなかったが，警察庁の自殺者数ページにデータ（といってもPDF）があった。CSVにして suicide.csv として置いておく（2021年は暫定値）。

import pandas as pd
import plotly.express as px

df = pd.read_csv("https://okumuralab.org/~okumura/python/data/suicide.csv")

fig = px.line(df, x='年', y=["男性", "女性"], markers=True, range_y=[0,26000])
fig.show()

1998年に不況のため特に男性の自殺が急増した（自殺の実態）。2020〜2021年は女性の自殺がおそらくコロナの影響で増えたが，男性はあまり変わらない。そういえば2011年の東日本大震災でも女性は微増したが男性はむしろ減少傾向が続いている。
