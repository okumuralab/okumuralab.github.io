実行

Python の実行には，いくつかの方法があります。

まず，Python のバージョンをチェックします：

python --version

これで Python 2.*.* と出るなら，次のコマンドを試します：

python3 --version

こちらのほうで Python 3.*.* と出るなら，以下では python や ipython の代わりに python3 や ipython3 としてください。

python コマンド（REPL）

ターミナルに

python

とだけ打ち込むと，>>> というプロンプトが現れますので，その右側に Python の命令を打ち込みます。例えば

print("こんにちは")

と打ち込めば，ターミナルに「こんにちは」と現れます。

もうちょっと長いものを打ち込んでみましょう：

for i in range(3):
    print("大事なので3回言います")

Python はインデント（字下げ）でブロック構造を表すので，インデントを省略しないでください。

ターミナルに戻るには，quit() または exit() と打ち込みます。

iPython

iPython なら，もう少し対話的に（interactive に）Python を使えます。ターミナルに

ipython

と打ち込みます（Emacs の M-x shell 内なら ipython --simple-prompt）。

ヘルプを見るには，例えば ?print のように打ち込みます（このヘルプの終了は q）。

quit と打ち込めば終了します。

スクリプト

Python プログラム，例えば

for i in range(3):
    print("大事なので3回言います")

は，ファイルに納めておくことができます。ファイル名は test.py のように拡張子を py にします。実行は

python test.py

と打ち込みます。

実行可能なスクリプトにするには，先頭に #! /usr/bin/env python を付けます：

#! /usr/bin/env python

for i in range(3):
    print("大事なので3回言います")

文字コードは UTF-8 にします（BOM を付けないようにします）。実行可能にするには，ターミナルで

chmod 755 test.py

のように打ち込みます。これで，ターミナルに

./test.py

と打ち込めば実行されます。

もしどうしても別の文字コード，例えば Shift JIS にしたいなら，

#! /usr/bin/env python
# -*- coding: shift_jis -*-

for i in range(3):
    print("大事なので3回言います")

のようにします。

Jupyter Notebook

ターミナルに

jupyter notebook &

と打ち込むと，Web ブラウザが立ち上がって，カレントディレクトリのファイル名一覧が表示されます。もし拡張子が ipynb のファイルがあれば，それを開けば前回の続きの計算ができます。新しいノートブックを開くには，この画面で [New ▼] ボタンを押し Python 3 を選びます。一番上の Untitled をクリックすると，ノートブックの名前が変えられます。

In [ ]: の右の箱に Python のコマンドを打ち込みます。何行かまとめて入れることもできます。実行するには Shift + Enter します。

[Code ↕︎] ボタンを [Markdown ↕︎] に切り替えれば，Markdown 形式で説明が書けます。$E = mc^2$ のように $ で挟んで数式を書くこともできます。

終了は File → Close and Halt です。右上の [Quit] で Web サーバが終了します。
