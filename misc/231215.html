機械学習・Deep Learningの無料本など

  • François Fleuret (2023): The Little Book of Deep Learning
  • Ian Goodfellow, Yoshua Bengio, Aaron Courville (2016): Deep Learning（ダウンロード不可、読むだけ）
  • Kevin Murphy: Probabilistic Machine Learning Book 1 (2022), Book 2 (2023) の draft がダウンロードできる
  • Christopher Bishop (2006): Pattern Recognition and Machine Learning (PRML)
  • Christopher Bishop with Hugh Bishop (2024): Deep Learning: Foundations and Concepts（ダウンロード不可、読むだけ）
  • Simon Prince (2023): Understanding Deep Learning
  • Zhang et al. (2023): Dive into Deep Learning
  • Jurafsky and Martin (2023): Speech and Language Processing
  • Deisenroth, Faisal, Ong (2020): Mathematics for Machine Learning
