Google ColabでGPU

MXNetである学習をしていたが，自分のMac miniではあまりにも遅いので，Coogle Colabで12時間まで無料で使えるGPUを試してみた。

「ランタイム」→「ランタイムのタイプを変更」でGPUを選ぶ。別マシンになるので，今までインストールしたものはすべて再インストールになり，計算も最初からやり直しになる。ここで間違えて pip install mxnet してしまったが，GPU版にしなければならないので pip uninstall mxnet で戻す。!nvcc --version するとCudaのバージョンが10.0と出た：

そこで

pip install mxnet-cu100

とする。これでMac miniで 14 examples/sec だった例が 801 examples/sec になった。

参考：Fast.ai Lesson 1 on Google Colab (Free GPU)，d2l.ai の Installation
