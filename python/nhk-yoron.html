NHK世論調査

NHK世論調査の中で政党の支持率などを問う政治意識月例調査 2017年の調査方式が変更（固定電話＋携帯電話）された4月以降の内閣支持率データを nhk-yoron.csv に収めた（文字コードUTF-8，行末LF，#以下はコメント）。これを読んでグラフを描いてみよう：

import pandas as pd

df = pd.read_csv("https://okumuralab.org/~okumura/stat/data/nhk-yoron.csv",
                 index_col='調査日', parse_dates=['調査日'], comment='#')
df.plot(marker="o", xlim=("2017-03-20","2019-07-31"))
plt.savefig('nhk-yoron.png', bbox_inches="tight")
