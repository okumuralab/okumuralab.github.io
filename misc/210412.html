CentOS 7にpython-popplerをインストール

小ネタ。pip install python-poppler がこけたので試行錯誤。

まず cmake のバージョンが古いと言っているみたい。yum install cmake3 で /usr/bin/{cmake3,ccmake3,cpack3,ctest3} を入れて，それを /usr/local/bin/{cmake,ccmake,cpack,ctest} にシンボリックリンクした（本当は alternatives というコマンドで cmake と cmake3 を切り替えるようにできるのだが手を抜いた）。

ほかには poppler poppler-cpp poppler-devel poppler-cpp-devel を yum install した。
