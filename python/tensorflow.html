TensorFlowによる機械学習

TensorFlow（テンサーフロー，テンソルフロー）は Google の機械学習ライブラリである。

TensorFlow 1 系（最新は 1.14）をインストールするには

pip install tensorflow  # または conda install tensorflow

新しい Tensorflow 2.0 beta をインストールするには（これをすると TensorFlow 1.x はアンインストールされる）：

pip install tensorflow==2.0.0-beta1

Google Colab で 2.x を使うには，インポートの前に

%tensorflow_version 2.x

と打ち込むようだ。

インポートする：

import tensorflow as tf

バージョンを確認する：

In [ ]: tf.__version__
Out[ ]: '2.0.0-beta1'    # または '1.14.0'

In [ ]: tf.keras.__version__
Out[ ]: '2.2.4-tf'

ごくごく簡単な例として，TensorFlow のチュートリアルの最初の例をやってみる。データには有名な MNIST を使う。これは28×28ピクセルの手書き画像で，ピクセル値は0から255までの整数である。

mnist = tf.keras.datasets.mnist

(x_train, y_train), (x_test, y_test) = mnist.load_data()

ピクセルは0〜255の整数値だが，0〜1の実数に変換する：

x_train = x_train / 255.0
x_test  = x_test  / 255.0

ごくごく簡単なモデルを定義する：

model = tf.keras.models.Sequential([
  tf.keras.layers.Flatten(input_shape=(28, 28)),
  tf.keras.layers.Dense(128, activation='relu'),
  tf.keras.layers.Dropout(0.2),
  tf.keras.layers.Dense(10, activation='softmax')
])

model.compile(optimizer='adam',
              loss='sparse_categorical_crossentropy',
              metrics=['accuracy'])

いよいよフィット（重い）：

model.fit(x_train, y_train, epochs=5)

最後に評価：

model.evaluate(x_test, y_test)

正解率 0.978 程度である。

上記ページのサンプルには載っていないが，予測は次のようにして求められる：

y_pred = model.predict_classes(x_test)

混同行列を求める：

tf.math.confusion_matrix(y_test, y_pred)

表示されない場合は次のようにする：

sess = tf.Session()
tf.math.confusion_matrix(y_test, y_pred).eval(session=sess)

学習済みのモデルをHDF5形式で保存する：

model.save("mymodel.h5")

保存したモデルは次のようにして読み出す：

model = tf.keras.models.load_model("mymodel.h5")
