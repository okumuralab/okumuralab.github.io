正規表現

正規表現を扱うための標準ライブラリ re（日本語ドキュメント）を使って正規表現に慣れましょう。

import re

if re.search("ほげ", "ほげほげ"):
    print("含まれます")
else:
    print("含まれません")

いろいろ変えてやってみましょう。

次のようなこともできます。

m = re.search("^ほげ(.*)$", "ほげふが")
m     # 一致しなければ偽
m[0]  # 一致全体
m[1]  # 1番目の括弧
