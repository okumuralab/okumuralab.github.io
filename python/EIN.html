EIN（アイン）

EIN -- Emacs IPython Notebook は Emacs 上の Jupyter Notebook クライアントです。Takafumi Arakaki さんが開発し，John Miller さんがメンテナンスされています。

インストールは melpa でできます。emacs.el に EIN 特有のことを書く必要はありません。

あらかじめ ~/.mailcap に Mac なら image/*; open -a Preview "%s" のように画像ビューアを設定しておきます（プレビューの環境設定でウィンドウの背景を白に設定しておく必要があります）。設定により画像をインライン表示することもできます。

起動は M-x ein:run するか .ipynb ファイルを開いて C-c C-o します。すでに起動している Jupyter サーバに接続するには M-x ein:login します（URLかポートを聞いてきますが，localhost:8888 で動いているなら 8888 と入れます）。終了は M-x ein:stop です。

セル（In [ ]: の下）に何行かのコードを入力して M-RET すると，実行され，次のセルが挿入されます。C-c C-a で上にセルを挿入，C-c C-b で下にセルを挿入します。C-u C-c C-a で上にマークダウンを挿入，C-u C-c C-b で下にマークダウンを挿入します。C-c C-k でセルを消します。C-c ↑ や C-c ↓ でセルを上や下に移動します。
