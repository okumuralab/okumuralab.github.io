EMNIST

MNISTを拡張したEMNIST：

  • The EMNIST Dataset
  • arXiv: EMNIST: an extension of MNIST to handwritten letters
  • PYPI: emnist 0.0

いくつか表示させてみる：

import emnist

images, labels = emnist.extract_training_samples('digits')

for k in range(24):
    plt.subplot(3, 8, k+1)
    plt.xticks([])
    plt.yticks([])
    plt.imshow(images[k], cmap='gray_r')
    plt.title(labels[k])
fig.savefig("emnistfig.png", bbox_inches="tight")

数字以外の文字：

images, labels = extract_training_samples('letters')
