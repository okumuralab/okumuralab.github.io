東京の死者数

Financial Times の COVID-19 グラフでお馴染みの超過死亡（excess mortality）が日本でも見られるかが話題になっている。すでにインフルエンザ関連死亡迅速把握システムに超過死亡が見えるという説もある（日本の「超過死」についての注意事項も参照）。しかし日本全体の人口動態統計速報は厚労省だから集計が遅い（2020年2月分は2020年4月24日に出た）。

そこで，日本で一番 COVID-19 の感染が深刻な東京都のデータを調べてみる。すでに何人もの人がグラフを描いているので，ここに書いたことはn番煎じである）。

東京都の死者数などの統計は東京都の人口（推計）以下にある。2011年の大震災の年も調べたかったので，2011年以降の毎月の死者数を tokyodeaths.csv というファイルに収めた。

import pandas as pd
import matplotlib.pyplot as plt

df = pd.read_csv('https://oku.edu.mie-u.ac.jp/~okumura/python/data/tokyodeath.csv')

単純に月ごとにプロットすると，大の月・小の月の影響で凸凹する。しかも，2020年のようなうるう年の2月は平年に比べて 3.6% もの「超過死亡」が見えてしまう。そこで，1日あたりの死者数に直してからプロットする。

def days(year, month):
    m = [31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31]
    return m[month - 1] + ((month == 2) and
                           ((year % 4 == 0) and (year % 100 != 0) or (year % 400 == 0)))
