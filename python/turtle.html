タートルグラフィックス

Python には turtle というタートルグラフィックスライブラリが標準で備わっています。対話型の環境で

from turtle import *

とすると、まだ何も起こりません。続けて

for i in range(5):
    forward(200)
    left(72)

と打ち込むと、Python Turtle Graphics という画面が開き、➤ のような「亀」が正五角形を描くはずです。

次の例は、ペンカラーを黒、フィルカラーを黄色にして、塗りつぶします。

color("black", "yellow")

begin_fill()
for i in range(5):
    forward(200)
    left(72)
end_fill()
