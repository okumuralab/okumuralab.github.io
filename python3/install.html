Python 3 インストール

いろいろな方法があります。私は今のところ Miniconda を使っています。

Python.org からのインストール

Download Python から，ご自分の環境に合った最新のインストーラをダウンロードし，インストールします。

Anaconda によるインストール

Anaconda Distribution からご自分の環境に合った最新のインストーラをダウンロードし，インストールします。何でも入りなので，たいへん大きいパッケージです。

Homebrew によるインストール（Mac）

brew install python で入れる方法と，brew install pyenv でまず pyenv を入れてから pyenv 経由で各種 Python を入れる方法などがあります。

Miniconda によるインストール

Miniconda から最新の Python 3.x 版インストーラをダウンロードし，実行します。私は Mac や CentOS で bash installer を使いました：

bash Miniconda3-latest-*-x86_64.sh

いろいろ聞いてくるので答えます。

パッケージ管理は conda コマンドで行います：

conda update --all
conda list
conda install ipython numpy scipy matplotlib pandas

ソースからのインストール

Download Python からソースのtarballをいただいてきて展開し，その中で次のように打ち込みます：

./configure
make
make test
make install

実行ファイルは /usr/local/bin に入りますので，そこにパスを通しておきます。

設定

matplotlib でグラフを描画するためには，日本語も使うなら，こちらから IPAex ゴシックをいただいてきてインストールし，~/.matplotlib/matplotlibrc に次のように書きます（もし古い ~/.matplotlib/fontlist*.json のようなファイルがあれば消しておきます）：

backend: TkAgg
font.family: IPAexGothic

PDF で保存する場合，IPAex ゴシックは Type 3 形式でサブセット埋め込みされます。Source Han Sans なども使えますが，サブセット化できず，丸ごと埋め込まれてしまい，ファイルサイズが巨大になります。

雛形の matplotlibrc の位置は Python で次のコマンドを実行すれば表示されます：

import matplotlib
matplotlib.matplotlib_fname()
