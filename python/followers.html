ツイッターのフォロワー数の推移をプロット

Rで描いたグラフの例：Twitterのフォロワー数のPython版である。

上と同様にしてソースからデータをファイルにコピーする。

var ar_data = [];
ar_data[1] = [...];
ar_data[2] = [...];
ar_data[3] = [...];
ar_data[4] = [...];
ar_data[5] = [...];
ar_data[6] = [...];
ar_data[7] = [...];
ar_data[8] = [...];
var ar_lbl = [];
ar_lbl[1] = [...];
ar_lbl[2] = [...];
ar_lbl[3] = [...];
ar_lbl[4] = [...];
ar_lbl[5] = [...];
ar_lbl[6] = [...];

ここでPythonなら var ar_data = []; を ar_data = [None] * 9，var ar_lbl = []; を ar_lbl = [None] * 7 に書き換えればそのまま使える。

import pandas as pd

t = [pd.to_datetime(x, format="%y%m%d") for x in ar_lbl[6]]
df = pd.DataFrame(index=t, data={'followers':ar_data[7]})
df.plot()

あとはいろいろ工夫する。例えば

df1 = df[df.index >= "2019-04-01"]
df1.plot(color="black", marker="o")
