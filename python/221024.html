6□4□5□2□1で0〜35を作る問題

6□4□5□2□1の□に+-×÷を入れて0〜35を作る問題をPythonで解いてみよう。まずは集合を使ってみる。

ex = ["6", "", "4", "", "5", "", "2", "", "1"]
op = ["+", "-", "*", "/"]

s = set()
for i in op:
    ex[1] = i
    for j in op:
        ex[3] = j
        for k in op:
            ex[5] = k
            for l in op:
                ex[7] = l
                e = "".join(ex)
                v = eval(e)
                if int(v) == v:
                    s.add(int(v))
