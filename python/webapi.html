Web API

文科省高等学校情報科「情報Ⅰ」教員研修用教材の第3章に簡単なWeb APIをPythonで叩く例が載っていたので，試してみる。使うのは株式会社アイビスが提供している郵便番号検索API。

import requests
import json

url = "http://zipcloud.ibsnet.co.jp/api/search"
param = {"zipcode": "5140007"}
res = requests.get(url, param)
data = json.loads(res.text)
address = data["results"][0]
print(address["address1"] + address["address2"] + address["address3"])

入力を含むプログラム：

#! /usr/bin/env python3

import requests
import json

url = "http://zipcloud.ibsnet.co.jp/api/search"
while True:
    zip = input('7桁の郵便番号を入力してください: ')
    if not zip:
        break
    print(zip)
    param = {"zipcode": zip}
    res = requests.get(url, param)
    data = json.loads(res.text)
    address = data["results"][0]
    print(address["address1"] + address["address2"] + address["address3"])
