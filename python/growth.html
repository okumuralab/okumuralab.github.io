増え方のシミュレーション

利息の増え方（複利計算）

預金 = 100000
利率 = 0.05
for i in range(11):
    print(i, 預金)
    預金 += 利率 * 預金

あるいは

預金 = [100000]
利率 = 0.05
for i in range(10):
    預金.append((1 + 利率) * 預金[-1])
預金

プロットするなら

import matplotlib.pyplot as plt

plt.plot(預金, "o-")

100年くらい預けたらいくらになるか。

生物も（少なくとも最初は）同じような増え方をする。

x = [1]
r = 0.05
for i in range(100):
    x.append(x[-1] * (1 + r))
plt.plot(x, "o-")

しかし，そのうちに上限に達し，それ以上増えなくなる。

xmax = 10
x = [1]
r = 0.05
for i in range(100):
    x.append(x[-1] * (1 + r * (1 - x[-1] / xmax)))
plt.plot(x, "o-")

このような曲線をロジスティック曲線という（ウィキペディアのロジスティック方程式参照）。
